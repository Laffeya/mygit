查看修改内容
git diff xx.txt

查看仓库当前的状态
git status

把修改提交到Git版本库
git add xx.txt
git commit -m "注释"

查看历史记录
git log
git log --pretty=oneline  简单一行显示

在版本的历史之间穿梭
git reset --hard commit_id   commit_id 一大串类似1094adb...
git reset --hard HEAD^
git reset --hard HEAD^^
git reset --hard HEAD~100
HEAD表示当前版本，上一个版本就是HEAD^，上上一个版本就是HEAD^^，当然往上100个版本写100个^比较容易数不过来，所以写成HEAD~100

查看命令历史
git reflog
